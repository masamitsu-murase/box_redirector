General

Box URL

Input the Box URL of the root folder. Typically, it should be https://app.box.com/folder/0, but it can be customized for enterprise environments.

Your access to opens the specified folder/file in Box web app.

Box Drive Redirection

You can configure a redirection URL and a parameter name. When the URL is accessed with the parameter which specifies a Box file or folder in your local drive, it will be redirected to the Box URL to open it in Box web app.

About Box Redirector

Overview

Box Redirector is a browser extension that enables seamless integration between Box Drive and the Box web application.

When you access a URL such as:

https://app.box.com/folder/0?box_redirector_path=C:/Users/username/Box/somefolder/sample.txt

Box Redirector automatically redirects you to the corresponding page for somefolder/sample.txt inside Box, allowing you to open files or folders in the Box web app directly from your local Box Drive path.
