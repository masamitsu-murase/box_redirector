Box Redirector について

概要

Box Redirector は、Box Drive と Box Web アプリケーションをシームレスに連携させるためのブラウザ拡張機能です。

例えば、以下のような URL にアクセスすると:

https://app.box.com/folder/0?box_redirector_path=C:/Users/username/Box/somefolder/sample.txt

Box Redirector は自動的に Box 内の somefolder/sample.txt のページにリダイレクトし、ローカルの Box Drive パスから直接 Box Web アプリでファイルやフォルダーを開くことができます。

注意: この拡張機能は Box のドキュメント化されていない API を利用しています。Box 側の仕様変更等により、動作しなくなる可能性があります。
